# 更新日志

## v0.3.6

- **精选推荐轮播优化**：改为连续滑动切换（旧的滑出、新的紧跟滑入，不再有中间空白/闪烁）；鼠标移上去时左右两侧出现切换按钮；底部居中触点可直接切到对应的精选
- **点精选横幅进入详情页**：点横幅任意处进详情，播放键仍单独播放（与其它卡片一致）

## v0.3.5

- **外部播放器也支持预选音轨/字幕**：用 **VLC**（Windows/Linux/macOS）或 **IINA**（macOS）播放时，详情页预选的音轨/字幕会自动切好，不必再进播放器里切（此前仅 mpv 支持）。PotPlayer / Infuse 因其自身不支持命令行选轨，仍需在播放器内切换（音轨/字幕选择器下方已注明）
- **继续观看卡片点击改为进入详情页**（不再直接播放）：想续播点卡片上的播放键或按空格；这样和其它卡片一致，也能先看详情再决定

## v0.3.4

- **系列电影自动聚合成合集**：按 TMDB 系列信息（而非文件夹）把系列电影（如指环王、霍比特人）归组成一个合集，**即使各部分散在不同文件夹也能聚合**；成员从浏览区收起、只在合集里出现（需填 TMDB API Key，刮削到系列信息后生效）
- **演员「发现作品」优化**：可按作品名搜索；文件源里**只有「已入库」的作品才可点开进详情**（有海报 / 简介 / 文件信息），未入库的置灰并标「未入库」（修复此前点进去没有视频信息的问题）
- **修复：用自己安装的 mpv 时，单独预选字幕 / 音轨不生效**（会被你 mpv 里的自动选轨脚本覆盖）——现在在文件加载后强制设回预选的轨道
- **首页精选轮播**：首页顶部精选横幅改为多部循环切换（自动切换 + 底部圆点手动切）
- **媒体库顶部媒体源按类型区分**：不同类型的源（Emby / Jellyfin / 本机 / WebDAV / DLNA…）用不同颜色的图标和类型标签区分
- **外部播放器 VLC**：能找到安装在非标准位置的 VLC（查注册表）；VLC 窗口大小交给它自己（单窗口）
- 横向行（演职人员 / 媒体源 / 分类等）改为只用左右按钮翻页，不再抢鼠标滚轮，避免影响上下滚动

## v0.3.3

- **文件源剧集分集显示真实信息**：用 TMDB 补充每集的真实集名 / 简介 / 剧照 / 时长（不再只是「第 N 集」；Emby / Jellyfin 本来就有，此项只补文件源）
- **修复系列电影被误当剧集**：如指环王三部曲这类每部各占一个文件夹的系列电影，不再被聚合成剧集；并在文件源剧集详情页新增「拆成电影」按钮，可手动把误识别的拆回单独的电影

## v0.3.2

- **剧集详情**：点某一集，简介 / 文件信息 / 音轨字幕会切换为该集的信息（在原详情页内切换，不跳转）；文件信息与音轨字幕加载时用骨架屏占位，不再一会有一会没有
- **编辑元数据**：「重新匹配」支持多个匹配项手动选择；匹配到多季剧集时可选具体某一季（用该季海报 / 简介 / 年份细化）；弹窗海报预览加宽
- **多季剧集聚合更准**：文件源里「爱情公寓 1~5」「剧名 第X季」（含中文数字）「剧名 / Season N」等能正确聚合成一部多季剧（番外篇 / 电影不会误并）
- 详情页背景图改用更高分辨率，不再模糊
- 使用自己安装的外部 mpv 播放时，保留你自己 mpv 的界面与配置（不再被自带界面覆盖；进度同步 / 续播 / 预选轨道仍生效）
- 单季剧集也显示「第 X 季」标签

## v0.3.1

- 修复：软件更新 / 重装后，最近添加的文件源（本机 / WebDAV / SMB / DLNA）与刚填的 TMDB API Key 可能丢失（媒体源 / 设置改为立即落盘 + 原子写入，避免更新时被强制关闭而丢数据）
- 优化：「媒体源」页里已停用的源不再可点击跳转到媒体库（信息置灰，开关 / 编辑 / 删除仍可用）

## v0.3.0

- **详情页音轨 / 字幕预选**：影片详情页可直接下拉选择音轨和字幕，播放时自动切好，不必再进播放器里切（Emby / Jellyfin / 文件源通用，仅 mpv）。Emby / Jellyfin 从服务器读取可选轨道；魔改 / 网盘版 Emby 读不到时，用自带 mpv 探测直连流兜底
- **点演职人员发现作品**：详情页点演员 / 导演，查看 TA 参演的其它作品（Emby / Jellyfin 为库内可播条目，文件源走 TMDB 展示）
- **Emby / Jellyfin 显示视频格式与文件位置**：详情页「文件信息」显示分辨率 / 编码 / HDR / 大小 / 码率 / 文件路径（公网魔改版用 mpv 探测兜底，先显示大小 / 路径再补齐格式）
- **外部播放器**：Windows 新增 VLC；macOS 找回 Infuse；未安装播放器时给出明确提示而非报错
- **界面**：标题栏不再显示图标与名称（只留窗口按钮），应用图标移到侧栏顶部；音轨 / 字幕选择器重新设计；演职人员 / 媒体源 / 分类等横向行支持鼠标滚轮左右滚动；演职人员卡片悬浮不再被裁切
- **播放更快**：mpv 按平台指定硬解码器 + 窗口立即弹出，明显缓解「点了半天才弹出播放器」；公网影片的格式 / 轨道信息加载也更快（少一次请求）
- 停用的媒体源不再出现在媒体库
- 关于页加上 GitHub 项目地址
- macOS 自带 mpv 播放窗口的红绿灯回到左上角

## v0.2.0

- 新增「文件浏览类源」：**本机存储 / WebDAV / SMB / DLNA**，无需服务器，视频直接并入媒体库
- 文件源视频：mpv 无头抽帧生成缩略图；支持「文件夹层级浏览」与「库视图」一键切换
- **TMDB 刮削**：在设置里填入 TMDB API Key 后，文件源视频自动匹配海报 / 简介 / 年份 / 评分 / 演职人员 / 相关推荐
- **剧集自动聚合**：把分集聚合成剧集（识别 `SxxExx` / `第x集` / 番剧命名，或按「剧名 / Season N」文件夹）；自动过滤蓝光原盘碎片与花絮 / 预告
- **手动编辑元数据**：自动刮削不准时，可在详情页编辑名称 / 海报 / 简介，或输入正确名「重新匹配」
- 播放增强（仅 mpv）：音轨 / 字幕语言预选、按章节名跳过片头片尾
- 界面：去掉原生标题栏、内容拉通到顶（自绘标题栏，左上角图标 + 名称，右上角窗口按钮跟随主题）；亮色下侧栏过渡优化
- 关于页图标换为应用图标，版本号跟随 `package.json` 自动同步
