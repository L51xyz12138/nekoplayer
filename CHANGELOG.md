# 更新日志

## v0.3.24

- **媒体库顶部改版**：原来的三行胶囊（媒体源 / 服务器分类 / 类型+搜索）合并为一条磨砂工具栏——媒体源与服务器分类收成下拉（当前选中项在按钮上回显）、类型筹码更紧凑，省出约 90px 首屏高度；工具栏常驻不随滚动消失，浏览海报墙时也能随时搜索切换
- **详情页文字更清晰**：背景大海报改为右上方清晰展示，左侧文字区用**渐进磨砂**（左强右弱的磨砂玻璃 + 薄色罩）替代原来的整片白色渐变——既能看清海报色彩、文字又始终可读，亮 / 暗主题都适用；「续看 S1E8 · 78%」进度整合成磨砂卡片
- **修复**：打开有「续看集」的剧集详情页时，页面会被自动拉到中间（本该只横向滚动剧集行定位到那一集，不该牵动整页垂直滚动）
- **直播频道墙改版**：频道卡改为白底台标 tile + 频道名外置卡片下方（不再压台标），多源角标保留；频道按分组归组展示、分组标题吸顶，滚动长清单时始终知道身在哪个分组
- **剧集卡标题更清爽**：刮到元数据的分集显示「第 N 集 · 真标题」；没刮到的（字幕组文件名）主标题只留「第 N 集」，原始文件名降级为弱化小字，不再一长串挤在标题里
- **继续观看更直观**：剧集卡片也标注剩余时长（如「S1E8 · 标题 · 剩 32 分钟」），电影剩余时长超过 1 小时改显「剩 X 小时 Y 分钟」
- **打磨**：统计页评分分布只画有评分的档位（不再刷一排 0 空条）；媒体源页停用行的置灰程度减轻（0.42→0.62，仍可读）；设置页 TMDB Key / assrt Token 默认密文显示、点眼睛可临时明文核对

## v0.3.23

- **修复**：整季连播时在 mpv 里跳集后，回到详情页点「播放」仍从跳集前那一集、旧进度接着播的问题——现在续看集前进后，「播放」按钮会跟随到新的续看集与进度（与页面显示一致；若你手动点选过某集，则仍尊重你的选择）

## v0.3.22

- **选源 + 选播放器**：点频道弹窗可选择播放源（同名频道的多个源自动归并，按主机名区分）和外部播放器（mpv / VLC / PotPlayer …）

- **台标兜底**：内嵌台标加载失败、或 TXT 清单没有台标时，按频道名从公共台标库补取，再退回彩色名字块
