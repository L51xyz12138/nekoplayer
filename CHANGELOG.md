# 更新日志

## v0.1.2

- 支持多个 Emby 媒体源聚合，持久化保存（刷新不再只剩一个）
- 播放进度续播：从上次位置继续；mpv 播放退出后进度自动同步回 Emby 并刷新页面
- 新增背景主题：亮色 / 暗色 / 跟随系统（默认跟随系统）
- 详情页显示播放进度条
- 库首页卡片可直接播放（剧集自动续看那一集）
- 演职人员支持鼠标滚轮横向滚动；卡片悬浮不再被裁切
- 兼容网盘挂载版 Emby 的播放（改用条目详情取媒体源）
- 默认画质选项新增 4K

## v0.1.1

- 修复网盘挂载版 Emby 无法播放
- macOS 版自带官方 mpv，开箱即用
- mpv 进程健壮化，修复关闭后再播放偶发打不开
- 媒体源弹窗支持 Esc 关闭；密码框加显示 / 隐藏按钮

## v0.1.0

- 首个版本：连接 Emby 服务器，浏览影视库
- 支持 mpv / IINA / VLC / Infuse / PotPlayer 外部播放
- 剧集整季连播、多主题强调色、详情页、演职人员
- GitHub Actions 三平台自动构建发布
