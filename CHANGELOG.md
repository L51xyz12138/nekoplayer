# 更新日志

## v0.1.9

- 新增键盘快捷键：`/` 聚焦搜索、方向键在卡片/剧集/详情按钮间移动、Enter 打开详情、Space 播放、Esc 失焦或返回

## v0.1.8

- 新增筛选器：媒体库可按「类型 / 年份 / 未看」筛选（顶栏「筛选」面板）
- 移除 Infuse：不再作为外部播放器选项，也不再以「Infuse 风格」描述
- 内部精简：移除内置 web 播放器与 hls.js 依赖，安装包体积更小

## v0.1.7

- 优化：播放结束后不再重新拉取整个媒体库，改为只刷新「继续观看」与当前条目进度，大幅降低请求量与流量（减轻公网服务器风控风险、消除播放后卡顿）

## v0.1.6

- 新增 Jellyfin 媒体源支持：添加媒体源时可选 Jellyfin，媒体库分类 / 合集 / 续播 / 进度同步等功能通用
- Plex 暂未实现，添加弹窗中标注为「开发中」并禁用，避免误连

## v0.1.5

- 新增媒体库分类浏览：直接读取服务器自带的媒体库（电影 / 动漫 / 纪录片…），可按库筛选；分类过多时分类行支持左右滚动
- 新增合集（Collection）支持：可搜索、首页有「合集」区，点进合集查看里面的影片（如霍比特人 1/2/3）
- 搜索改为跨全部媒体库与类型，能搜到剧集、合集
- 主页 / 详情 / 设置页显示可见的纵向滚动条（深浅主题都清晰）
- 修复：卡片鼠标悬浮上移时上边缘被裁剪（剧集单集、最近添加行）

## v0.1.4

- 大媒体源优化：添加源后弹窗立即关闭、库在后台加载并显示「正在拉取媒体信息」提示
- 媒体库改为增量渲染（滚动到底自动加载更多），大库滚动、切换电影 / 剧集更流畅
- 删除媒体源前增加二次确认
- mpv 音轨默认改为原声（番剧走日语、欧美片走英语，不再强制中文配音），字幕仍中文优先
- 修复：亮色主题下左侧栏当前项文字看不清
- 搜索框增加一键清除按钮

## v0.1.3

- 新增「标记已看 / 未看」，详情页按钮点亮、卡片显示已看角标
- 媒体库加载改为骨架屏 + 启动秒开：缓存上次结果先渲染，再后台刷新
- 继续观看更智能：精准定位到正在看的那一集与进度，并按最近播放时间排序
- 详情页自动定位并高亮到续看的那一集
- 修复：添加媒体源弹窗未跟随亮 / 暗主题
- 修复：软件更新后媒体源 / 设置丢失（改用本地文件存储，不再受浏览器存储限制）
- 修复：续播未定位到进度、总是从每集开头播放
- 修复：详情页剧集列表在刷新 / 播放结束后消失
- 修复：继续观看选错集、顺序不随最近播放更新
- 兼容魔改 / 网盘挂载版 Emby 的进度字段差异

## v0.1.2

- 支持多个 Emby 媒体源聚合，持久化保存（刷新不再只剩一个）
- 播放进度续播：从上次位置继续；mpv 播放退出后进度自动同步回 Emby 并刷新页面
- 新增背景主题：亮色 / 暗色 / 跟随系统（默认跟随系统）
- 详情页显示播放进度条
- 库首页卡片可直接播放（剧集自动续看那一集）
- 演职人员支持鼠标滚轮横向滚动；卡片悬浮不再被裁切
- 兼容网盘挂载版 Emby 的播放（改用条目详情取媒体源）
- 默认画质选项新增 4K

## v0.1.1

- 修复网盘挂载版 Emby 无法播放
- macOS 版自带官方 mpv，开箱即用
- mpv 进程健壮化，修复关闭后再播放偶发打不开
- 媒体源弹窗支持 Esc 关闭；密码框加显示 / 隐藏按钮

## v0.1.0

- 首个版本：连接 Emby 服务器，浏览影视库
- 支持 mpv / IINA / VLC / PotPlayer 外部播放
- 剧集整季连播、多主题强调色、详情页、演职人员
- GitHub Actions 三平台自动构建发布
